Open Source

With ERPNext you have "full freedom" to do what you want with it. You can host it on your own and customize it to your requirements. ERPNext is built on a powerful web application framework based on Python and Javascript, Frappe. With Frappe, you can easily extend ERPNext by using the API or creating fully featured extensions (called apps) implementing custom functionality for your business.
